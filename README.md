# linux-docker-project
리눅스 도커를 이용한 초간단 웹사이트

# 개요
- xshell을 이용해 학교 서버 접속
- 도커를 이용해 아파치 이미지 다운
- 자신이 만들고 싶은 간단 웹사이트 만들기
- 보고서 작성
